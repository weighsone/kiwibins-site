IMAGES REQUIRED
===============

Place the following image files in this directory before deploying:

1. og-default.png  (1200×630px)
   Open Graph / social sharing image.
   Suggested content: Kiwibins logo + "Levin's Local Bin Company" text on green background.
   Referenced in: src/components/Layout.astro

To create og-default.png quickly you can use Canva, Figma, or any image editor.
Recommended dimensions: 1200 × 630 pixels, PNG or JPG format.

The site will still build and work without this file, but social media previews
will not show an image when the site is shared.
